# An Exercise Tracking App

[See it Here](https://brynn-exercise-tracker.herokuapp.com/)

this is a full stack application designed to record workout sessions for a user. The front end was created using ReactJS and bootstrap, and the backend was using NodeJS. The data is stored in a cloud-based DB; Mongo Atlas. The final product is deployed on Heroku.

**The Stack:**
* React, Bootstrap
* ExpressJS - Node
* MongoDB
